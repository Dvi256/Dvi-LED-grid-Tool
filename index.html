Sending Cards +

Signal capacity is calculated from pixel load per sending-card Ethernet port / main signal cable. The DVI default is 491,520 pixels per main signal run, which derives 30 panels at 128 x 128 px or 15 panels at 128 x 256 px. One main signal cable feeds the first panel; remaining panels use panel-to-panel loop cables.

Power Management +

Power capacity is calculated from pixel-derived watt load per main power cable. The DVI field default uses a 262,144-pixel power budget, which derives 16 panels at 128 x 128 px or 8 panels at 128 x 256 px when using the same reference wattage. One main power cable feeds the first panel; remaining panels use panel-to-panel power loop cables. Final distribution and compliance must be confirmed locally.
